# Avansas Case Study

Basic user management system with Spring Boot & JSP.

#### Used Technologies

----

- Spring Data JPA
- Spring Web
- PostgreSQL
- Spring Security
- Hibernate Validation
- Swagger UI
- Mapstruct
- Guava
- JSTL
- JQuery
- Bootstrap
